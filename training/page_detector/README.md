# Page Detector Training Scaffold

This folder contains the scaffolding to train a custom YOLOv8/YOLO11 segmentation model that specifically targets A4 document pages.

## Dataset Preparation

1. Gather images of A4 documents on various desks/backgrounds.
2. The hard-negative collection generated by the server (`server/data/page_detector_review/`) is a great place to start!
3. Annotate the images with **polygons** (segmentation masks), NOT bounding boxes. Label them as class `0: document_page`.
4. Place the dataset in the root of the project like so:

```
datasets/a4_page/
    images/
        train/
        val/
    labels/
        train/
        val/
```

## Training

Ensure your virtual environment is activated and `ultralytics` is installed.

Make the script executable and run it:
```bash
chmod +x training/page_detector/train_yolo_seg.sh
./training/page_detector/train_yolo_seg.sh
```

## Exporting Weights

Once training completes, export the best weights to the server folder so the `PageDetector` class picks them up automatically:

```bash
chmod +x training/page_detector/export_yolo_seg.sh
./training/page_detector/export_yolo_seg.sh
```

Restart the Mac server after exporting.
